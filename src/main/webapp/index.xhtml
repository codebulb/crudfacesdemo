Project status

Note that until 1.0 release, every component of this framework should be considered “experimental”. On the individual component’s pages, labels represent the current status of the component:

  • : Still experimental, but the main implementation work is considered “done”. Potential new functionality may be added in future releases.
  • : Offers enough functionality to use it experimentally, but still lacks non-basic features which are currently under development.
  • : Offers only a bare minimum of functionality. Is hardly of any use in a real-world environment until more development work is done.
