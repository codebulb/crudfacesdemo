CrudEntity
    • An abstract generic base class for a persistent business entity (model).

    Use either the CrudIdentifiable interface or the CrudEntity class to derive your entity model classes from. This is the only prerequisite to use them with a CrudService and a CrudController.

    The difference between the interface and the class is that the latter provides an auto-generated Long id field implementation out-of-the-box.

    For this demo application, a Customer entity is created by deriving from CrudEntity like so:

    @Entity
public class Customer extends CrudEntity {
    @NotNull
    private String firstName;
    @NotNull
    private String lastName;
    
    public String getInitials() {
        return String.valueOf(firstName.charAt(0)) + String.valueOf(lastName.charAt(0));
    }

    public String getFirstName() {
        return firstName;
    }

    public void setFirstName(String firstName) {
        this.firstName = firstName;
    }

    public String getLastName() {
        return lastName;
    }

    public void setLastName(String lastName) {
        this.lastName = lastName;
    }
}

    Apart from the @Entity annotation, this class consists solely of business logic.
